# Project Abstract

**Title:** Real-Time Network Traffic Monitoring & Packet Analysis  
**Author:** C. Pravin Sai (2420030777) | S-11  
**Course Instructor:** Chandusha Kanda

---

This project presents the design and implementation of a real-time network monitoring system using native Kali Linux tools. Modern networks generate vast streams of data every second — without continuous visibility, performance degradation and security threats can go undetected, often until it is too late.

The project demonstrates how **nload**, **tcpdump**, and **ping** can be combined to monitor live bandwidth usage, capture packets at the data-link layer, and analyse communication patterns between networked devices — all without requiring enterprise hardware or proprietary software.

The system follows a modular, four-layer architecture: Traffic Generation → Traffic Monitoring → Packet Capture → Analysis. This approach enables real-time visibility into network behaviour, identification of source/destination IPs and protocols, and detection of anomalous traffic patterns.

The project highlights fundamental computer network concepts including bandwidth monitoring, packet inspection, ICMP communication, and interface-level traffic analysis — reflecting practical, accessible network security and monitoring strategies.

> *Note: This project was developed as part of the CN Hackathon for the Computer Networks course.*
